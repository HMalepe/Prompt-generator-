Prompt Forge

Command-line tool, not a website

This repository is a Python CLI that forges dated trees of premium, zero-filler AI prompt packs and exports any leaf to Word + PDF. There is no web front-end to deploy.

Run it locally: python forge.py build --category websites --dry-run

This page exists only so the Vercel deployment succeeds instead of trying to build a Next.js app. See README.md.
